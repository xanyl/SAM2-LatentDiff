# Colab Notebooks

Place your Google Colab notebooks (.ipynb) here:

- `01_Setup_and_Download.ipynb` — Environment setup, dataset download
- `02_Preprocessing.ipynb` — SAM2 feature extraction, VAE encoding
- `03_Train_Stage1.ipynb` — Latent-space LoRA training (05d notebook)
- `04_Train_Stage2.ipynb` — PixelRefiner training (05h notebook)
- `05_Evaluate.ipynb` — Full evaluation + ablation studies

These notebooks correspond to the scripts in `scripts/` but are
configured for Google Colab Pro with A100 GPU.
